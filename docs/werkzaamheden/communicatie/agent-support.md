## Veelvoorkomende vragen van agenten

| Vraag | Actie |
|-------|-------|
| "Ik kan niet inloggen in Perfectview" | Check of licentie actief is, reset wachtwoord via [Abonnement](../../perfectview/abonnement.md) |
| "Hoe maak ik een verkoopkans aan?" | Verwijs naar [Verkoopkansen & Fasen](../../perfectview/verkoopkansen.md) |
| "De listing staat niet goed op de website" | Check WordPress, zie [Listing aanmaken](../../wordpress/listing-aanmaken.md) |
| "Klant wil een brochure" | Verwijs naar [Brochure aanvraag](../../perfectview/brochure-aanvraag.md) |
| "Ik heb een deal gesloten, wat nu?" | Factuur aanmaken, zie [QuickBooks](../administratie/quickbooks.md) |
| "Eigenaar wil OTO-contract opzeggen" | Verwijs naar [Contracten](../../perfectview/contracten.md) |

## Escalatie

Zaken die niet zelf op te lossen zijn:

| Probleem | Escaleer naar |
|----------|--------------|
| Technisch probleem website | Jan Barendse (zie [Contactpersonen](../contactpersonen.md)) |
| Juridische vraag | Advocaat / juridisch adviseur |
| Financieel geschil | Management |
| Klacht van klant | Management + betrokken agent |

## Tips

!!! tip "Beschikbaarheid"
    Wees bereikbaar tijdens kantooruren (8:00–17:00 AST). Buiten kantooruren alleen voor urgente zaken.

!!! tip "Handleidingen eerst"
    Verwijs agenten eerst naar de handleidingen-website voordat je zelf gaat uitleggen. Dit stimuleert zelfredzaamheid.
