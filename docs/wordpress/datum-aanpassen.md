# Stap 11: Datum aanpassen

Door de publicatiedatum van een listing aan te passen kun je deze **hoger op de website plaatsen**. Listings worden standaard gesorteerd op datum, dus een recentere datum = hogere positie.

## Wanneer de datum aanpassen?

- Als een listing meer aandacht nodig heeft
- Na een prijsverlaging
- Na een belangrijke update van de listing
- Om een listing weer bovenaan te plaatsen

## Hoe de datum aanpassen?

![Datum aanpassen](../images/wordpress/Website datum aanpassen \(omhoog plaatsen website\).png)

1. Open de listing die je omhoog wilt plaatsen
2. Klik rechts op **"Bewerken"** naast de publicatiedatum
3. Stel de datum in op **vandaag** (of de gewenste datum)
4. Klik op **"OK"**
5. Klik op **"Bijwerken"** om de wijziging op te slaan

!!! info "Alternatief"
    Je kunt ook via **"Listing gegevens"** de datum beheren. Dit is een betere methode voor het plannen van automatische date-updates.

## Effect

Na het aanpassen van de datum:

- De listing verschijnt hoger in de overzichtspagina's
- De listing kan opnieuw verschijnen in de "Nieuw" sectie
- Google registreert de update

!!! warning "Gebruik met mate"
    Pas de datum niet te vaak aan voor dezelfde listing. Dit kan onnatuurlijk overkomen voor bezoekers.
