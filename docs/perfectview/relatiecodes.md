# Relatiecodes & Labels

Dit is een naslagpagina met alle relatiecodes en labels die At Home Curaçao gebruikt in Perfectview.

## Relatiecodes

Wijs bij elk contact de juiste relatiecodes toe:

| Code | Betekenis | Wanneer toewijzen |
|------|-----------|-------------------|
| **KLS** | Koop Laag Segment | Koopinteresse < XCG 500.000 |
| **KHS** | Koop Hoog Segment | Koopinteresse > XCG 500.000 |
| **HLS** | Huur Laag Segment | Huurinteresse < XCG 2.500/mnd |
| **HHS** | Huur Hoog Segment | Huurinteresse > XCG 2.500/mnd |
| **VKV** | Vakantieverhuur | Interesse in vakantieverhuur |
| **BOG** | Bedrijfsonroerend goed | Interesse in commercieel vastgoed |
| **INV** | Investeringspanden | Interesse in vastgoedinvestering |
| **MAK** | Makelaar | Partner-makelaar |
| **HYP** | Hypotheek | Hypotheek-gerelateerd contact |
| **KAV** | Kavel Koop | Interesse in bouwkavels |
| **TAX** | Taxatie / Waardebepaling | Taxatie- of waardebepaling aanvraag |
| **EMI** | Emigratie begeleiding | Interesse in emigratie naar Curaçao |

!!! info "Meerdere codes mogelijk"
    Een contact kan meerdere relatiecodes hebben. Bijvoorbeeld: iemand die zowel wil kopen (KHS) als investeren (INV).

## Listing labels (WordPress)

Elke listing op de website krijgt een statuslabel:

| Label | Kleur | Wanneer |
|-------|-------|---------|
| **Nieuw** | Paars | Nieuwe listing (eerste 2-4 weken) |
| **Verlaagd** | Groen | Prijs is verlaagd |
| **Verkocht** | Rood | Pand is verkocht |
| **Verhuurd** | Oranje | Pand is verhuurd |
| **Optie** | Fuchsia | Er is een optie genomen |
| **Tip** | Geel | Aanbevolen door At Home |
| **Te Koop** | Roze | Standaard koopwoning |
| **Te Huur** | Blauw | Standaard huurwoning |
| **Exclusief** | Blauw | Exclusief bij At Home |

## Verkoopkans fasen

De fasen van een verkoopkans in Perfectview:

| Fase | Omschrijving | Kans % |
|------|-------------|--------|
| 1 | Gegevens aan AHC agent doorgegeven | 5% |
| 2 | Bezichtiging gepland | 15% |
| 3 | Bezichtiging uitgevoerd | 25% |
| 4 | Onderhandeling | 50% |
| 5 | Optie genomen | 75% |
| 6 | Contract getekend | 90% |
| 7 | Afgerond / Verkocht | 100% |
