# Procedures & Afspraken

Deze pagina bevat de werkafspraken en procedures voor het verwerken van listings in WordPress en Perfectview.

## Procedure nieuwe listing

### Stap 1: Contract

1. Maak een **OTO-contract** (Opdracht Tot Verkoop) met de eigenaar
2. Gebruik de template op Google Drive (beschikbaar in NL en EN)

### Stap 2: Listing plaatsen

- Plaats de nieuwe listing **binnen 2 weken** op WordPress
- Bij "stille verkoop": gebruik status **"Privé"** of **"Concept"**

### Stap 3: Agent verantwoordelijkheden

- Update bestaande listings met het **"Nieuw"** label
- Je mag de listing van een collega **niet** op "Concept" zetten zonder:
    1. Minimaal **2 waarschuwingen** in het weekoverleg
    2. Een **e-mail notificatie** aan de betreffende collega
- Stuur het objectnummer per e-mail naar de listinghouder

### Stap 4: Eigenaar- en courtagegegevens

**In WordPress listing:**

- Naam eigenaar
- Courtage/bemiddelingskosten afspraken

**In Perfectview:**

- Volledige eigenaargegevens (naam, adres, telefoon, e-mail)
- OTO-contract
- Courtage-afspraken

### Stap 5: Gevoelige gegevens

Gevoelige eigenaarinformatie wordt bijgehouden in een **apart spreadsheet** (alleen toegankelijk voor directie en listinghouder):

| Veld | Beschrijving |
|------|-------------|
| Listing nummer | Objectnummer van de listing |
| Lidnummer | Perfectview lidnummer |
| Bedrijfsnaam | Naam van het bedrijf |
| Lidnaam | Naam van de eigenaar |
| Telefoon | Telefoonnummer |
| E-mail | E-mailadres |

### Stap 6: Teamcommunicatie

- Nieuwe listings en belangrijke wijzigingen: melden in het **weekoverleg** + **e-mail** aan alle collega's

## Relatiecodes in Perfectview

Gebruik de juiste relatiecodes bij het aanmaken van contacten:

| Code | Betekenis |
|------|-----------|
| **KLS** | Koop Laag Segment (< XCG 500.000) |
| **KHS** | Koop Hoog Segment (> XCG 500.000) |
| **HHS** | Huur Hoog Segment (> XCG 2.500/mnd) |
| **HLS** | Huur Laag Segment (< XCG 2.500/mnd) |
| **VKV** | Vakantieverhuur |
| **BOG** | Bedrijfsonroerend goed (huur/koop) |
| **INV** | Investeringspanden |

## Begrippen en afkortingen

| Term | Betekenis |
|------|-----------|
| **Kandidaat/Prospect** | Potentiële klant met bekende interesse |
| **OTO** | Opdracht Tot Verkoop |
| **Objectnummer** | Uniek nummer voor de listing |
| **Klant** | Iemand die eerder heeft gehuurd/gekocht |
| **Suspect** | Potentiële toekomstige klant |
| **Agent** | Direct medewerker van At Home Curaçao |
| **Partner** | Samenwerkende partner-makelaar |
| **Courtage** | Makelaarscourtage (% van verkoopsom) |
| **Listing** | Vastgoedadvertentie |
| **PV/CRM** | Perfectview (klantbeheersysteem) |
| **Lead** | Aanvraag of verkoopkans |
| **Lid** | Vastgoedeigenaar (persoon of bedrijf) |
| **OG** | Onroerend goed |
| **BOG** | Bedrijfsonroerend goed |
| **Verkoopkans** | Verkoopkans met specifiek objectnummer in PV |
| **Zoekopdracht** | Zoekvraag zonder specifiek pand |
