## Periodieke taken

### Maandelijks

- [ ] Maandcijfers aanleveren aan accountant (omzet, kosten, facturen)
- [ ] Declaraties indienen (zie [Declaraties](../administratie/declaraties.md))
- [ ] Bankafschriften controleren

### Per kwartaal

- [ ] BTW/OB-aangifte voorbereiden (of door accountant laten doen)
- [ ] Kwartaaloverzicht commissies per agent

### Jaarlijks

- [ ] Jaarrekening laten opstellen door accountant
- [ ] Belastingaangifte voorbereiden
- [ ] Beoordelen of abonnementen/contracten verlengd moeten worden

---

## Wat moet je aanleveren aan de accountant?

| Document | Frequentie | Format |
|----------|-----------|--------|
| Verkoopfacturen | Maandelijks | PDF uit QuickBooks |
| Inkoopfacturen/bonnen | Maandelijks | Scan/foto |
| Bankafschriften | Maandelijks | PDF van bank |
| Commissieoverzicht per agent | Per kwartaal | Excel |
| Declaraties | Maandelijks | Excel |

---

## Tips

!!! tip "Vaste momenten"
    Plan een vast moment per maand (bijv. eerste vrijdag) om alle stukken voor de accountant te verzamelen en te versturen.

!!! warning "Bewaren"
    Bewaar alle financiële documenten minimaal 7 jaar (wettelijke verplichting Curaçao).
