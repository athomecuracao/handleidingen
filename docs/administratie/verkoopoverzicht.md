# Verkoopoverzicht

Het verkoopoverzicht is een Excel-sheet waarin alle gerealiseerde verkopen, verhuur en verwachte transacties worden bijgehouden.

## Structuur van het overzicht

Het bestand bevat drie secties:

### 1. Verkoop resultaten

Hier worden alle **gerealiseerde** transacties genoteerd.

| Kolom | Wat invullen |
|-------|-------------|
| **Makelaar** | Initialen: JD, JB, SW, KW |
| **Datum** | Datum van de transactie (dd.mm.jjjj) |
| **Objectnummer** | Objectnummer uit de listing (bv. #1434656) |
| **Omschrijving object** | Korte omschrijving (bv. "Cas Grandi") |
| **Type** | Verkoop: **KLS** (Koop Laag Segment) of **KHS** (Koop Hoog Segment) / Verhuur: **HLS** (Huur Laag Segment) of **HHS** (Huur Hoog Segment) |
| **Partner/Makelaar** | Naam samenwerkende partner (indien van toepassing) |
| **Bedrag XCG** | Verkoopprijs of huurbedrag in XCG |
| **Bemiddelingsfee %** | Courtage percentage |
| **Totaal XCG** | Wordt automatisch berekend (bedrag x fee%) |
| **Aanvang huur / Transport** | Datum aanvang huur of datum transport |
| **Verhuurders / Verkopers** | Naam verhuurder of verkoper |

### 2. Outlook komende 2 weken

Hier worden **verwachte** transacties voor de komende 2 weken genoteerd. Dezelfde kolommen als de verkoopresultaten.

!!! tip "Vergadering"
    Deze sectie wordt besproken tijdens de maandelijkse vergadering (zie [Agenda Vergadering](agenda-vergadering.md)).

### 3. Financiele planning

Overzicht van verwachte betalingen en transporten. Dezelfde kolommen als hierboven.

## Hoe invullen?

1. Open het bestand `Verkoopoverzicht At Home Curacao 2026.xlsx`
2. Ga naar het tabblad van het huidige jaar
3. Vul een nieuwe rij in bij de betreffende sectie
4. Het **Totaal XCG** wordt automatisch berekend
5. Sla het bestand op

!!! warning "Belangrijk"
    Vul het verkoopoverzicht **direct** bij na elke gerealiseerde transactie. Dit overzicht wordt gebruikt tijdens de maandelijkse vergadering.
