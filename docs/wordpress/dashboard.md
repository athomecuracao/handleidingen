# Stap 2: Dashboard & Backend

Na het inloggen zie je het WordPress Dashboard. Dit is het startpunt voor al je werkzaamheden.

## Het Dashboard

![WordPress Dashboard](../images/wordpress/Wordpress Dashboard At Home Curacao website.png)

Het dashboard toont een overzicht van:

- Websitestatistieken en bezoekersaantallen
- Recente activiteiten en blogposts
- Snelle links naar veelgebruikte functies

## Het Backend — Property Overzicht

Klik in het linkermenu op **"Properties"** om het overzicht van alle listings te zien.

![WordPress Backend](../images/wordpress/Wordpress backend At Home Curacao website.png)

Hier zie je alle property listings in een tabel met:

| Kolom | Beschrijving |
|-------|-------------|
| **Titel** | Naam van de listing |
| **Thumbnail** | Kleine preview-afbeelding |
| **Status** | Gepubliceerd, Concept, of Prullenbak |
| **Datum** | Publicatiedatum |
| **Categorieën** | Type vastgoed |

## Belangrijke menu-items

| Menu-item | Functie |
|-----------|---------|
| **Properties** | Alle vastgoedlistings beheren |
| **Properties → Add Property** | Nieuwe listing aanmaken |
| **Berichten** | Blogartikelen beheren |
| **Media** | Afbeeldingen en bestanden |
| **Master Slider** | Homepage slider instellen |

## Admin Columns — Uitgebreid overzicht

Met de Admin Columns plugin heb je een uitgebreid overzicht van alle listings met extra kolommen.

![Admin Columns overzicht](../images/wordpress/Word[ress back end Admin Collums gebruik.png)

Dit overzicht toont naast de standaard kolommen ook:

| Kolom | Beschrijving |
|-------|-------------|
| **Afbeelding** | Thumbnail van de listing |
| **Auteur** | De agent die de listing heeft aangemaakt |
| **Categorieën** | Type vastgoed (Koop, Huur, BOG, etc.) |
| **Property Type** | Woning, Villa, Appartement, etc. |
| **Featured Image** | Of er een uitgelichte afbeelding is ingesteld |
| **Prijs** | De vraagprijs |
| **Status** | Gepubliceerd, Concept, Offline, etc. |

!!! tip "Filteren en zoeken"
    Gebruik de **filters** bovenaan (Alle data, Auteur, Categorie, Type) om snel specifieke listings te vinden. Gebruik het **zoekveld** rechtsboven om op titel of objectnummer te zoeken.
