# Stap 1: Inloggen op WordPress

## Ga naar de website

1. Open je webbrowser en ga naar: **athomecuracao.com**
2. Klik linksboven op **"At Home Curacao Makelaars"** om naar het beheergedeelte te gaan

Of ga rechtstreeks naar:

```
https://athomecuracao.com/wp-login/
```

## Vul je gegevens in

1. **Gebruikersnaam** — Vul je gebruikersnaam in die je van je beheerder hebt ontvangen
2. **Wachtwoord** — Vul je wachtwoord in
3. **2FA-code** — Vul de 6-cijferige code in uit je **Google Authenticator** app
4. Vink eventueel **"Onthoud mij"** aan als je op je eigen computer werkt
5. Klik op **"Inloggen"**

!!! warning "2FA vereist"
    Voor het inloggen op WordPress is **tweefactorauthenticatie (2FA)** via Google Authenticator vereist. Zorg dat je de Google Authenticator app op je telefoon hebt geinstalleerd en gekoppeld aan je WordPress-account. Vraag de beheerder als je hulp nodig hebt bij het instellen.

!!! warning "Let op"
    Vink "Onthoud mij" **niet** aan op een gedeelde of openbare computer.

## Na het inloggen

Na succesvol inloggen kom je op het **WordPress Dashboard** terecht. Ga verder naar [Stap 2: Dashboard & Backend](dashboard.md).

!!! danger "Wachtwoord vergeten?"
    Klik op "Wachtwoord vergeten?" op het inlogscherm, of neem contact op met je beheerder.
